# IOS Data Room Standards

Standards for data room construction, organization, and document management in InvestOS workflows.

## Data Room Structure

### 21+ Document Categories
1. **Executive Summary** - Company overview and investment highlights
2. **Corporate Documents** - Articles, bylaws, operating agreement
3. **Financial Statements** - Historical and projected financials
4. **Financial Model** - Complete model with assumptions
5. **Valuation Analysis** - Valuation report and methodology
6. **Cap Table** - Current ownership and round modeling
7. **Legal Structure** - Entity diagram and jurisdiction info
8. **Regulatory Filings** - SEC filings, state registrations
9. **PPM / Offering Circular** - Primary offering document
10. **Subscription Agreement** - Investor subscription docs
11. **Risk Factors** - Comprehensive risk disclosure
12. **Market Research** - TAM/SAM/SOM and market analysis
13. **Competitive Analysis** - Competitive landscape
14. **Management Team** - Bios, resumes, references
15. **Board of Directors/Advisors** - Board composition and advisor network
16. **Intellectual Property** - Patents, trademarks, trade secrets
17. **Customer/Revenue Data** - Contracts, pipeline, metrics
18. **Operational Blueprint** - SOPs, processes, org chart
19. **Technology Overview** - Architecture, security, scalability
20. **Insurance** - Coverage summaries
21. **Tax Documentation** - Tax returns, structure optimization

### Organization Standards
- Numbered folder hierarchy matching categories above
- Consistent file naming: `{category-number}-{document-name}-{version}-{date}.pdf`
- Index document at root level with links to all documents
- Version control with clear current vs. archived distinction

## Quality Standards
- All documents must be current (within 90 days unless historical)
- Cross-reference validation must pass before data room is shared
- No placeholder or draft documents in shared data room
- All financial figures must be consistent across documents
- Sensitive documents may have restricted access tiers
