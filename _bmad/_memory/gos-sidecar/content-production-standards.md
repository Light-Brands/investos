# GOS Content Production Standards

Standards for content deliverables across all GrowthOS content workflows.

## SEO Content Standards

### Keyword Targeting Requirements
- Primary keyword must be specified for every SEO content piece
- Secondary keywords (3-5) must be listed with search volume data
- Keyword difficulty score must be documented
- Search intent classification (informational, navigational, transactional, commercial)
- Target SERP feature (featured snippet, People Also Ask, video carousel)

### On-Page Optimization
- Title tag with primary keyword (under 60 characters)
- Meta description with primary keyword and CTA (under 160 characters)
- H1 contains primary keyword, H2s contain secondary keywords
- Internal linking to minimum 3 related pages
- External linking to minimum 2 authoritative sources

## Funnel Stage Mapping

### Required Classification
- Every content piece must be tagged with funnel stage (TOFU, MOFU, BOFU)
- Content must include appropriate CTA for its funnel stage
- TOFU: Awareness-level CTAs (subscribe, follow, read more)
- MOFU: Consideration-level CTAs (download, register, compare)
- BOFU: Decision-level CTAs (buy, schedule demo, start trial)

## Brand Voice Consistency

### Voice Requirements
- All content must align with established product brand voice guidelines
- Tone adjustments for channel are permitted but core voice must be maintained
- Terminology must be consistent across all content (use approved glossary)
- Brand-specific phrases and taglines must be used correctly

## CTA Standards

### Tracking Requirements
- Every CTA must use trackable UTM parameters
- CTA copy must be action-oriented and specific
- Each content piece must have exactly one primary CTA
- Secondary CTAs are permitted but must not compete with primary
- CTA placement must follow content type best practices

## Channel-Appropriate Formatting

### Format Requirements
- Blog: Long-form with headers, images, and scannable structure
- Social: Platform-native formatting (character limits, hashtags, aspect ratios)
- Email: Mobile-first design, single-column, clear CTA above fold
- Video: Hook in first 3 seconds, captions required, end screen CTA
- Landing Pages: Above-fold value proposition, social proof, single conversion path
