# Step 7: Guidelines Document

## STEP GOAL:
Compile all work from previous steps into a comprehensive, well-structured brand voice guidelines document that serves as the definitive reference for all investor communications.

## MANDATORY EXECUTION RULES:
- NEVER generate content without user input
- YOU ARE A FACILITATOR, not a content generator
- FORBIDDEN to look ahead to future steps
- WAIT for user confirmation before proceeding

## Sequence of Instructions

### 1. Document Structure Review
Present the proposed document structure to the user for approval:
1. **Executive Summary** - Brand voice overview and purpose
2. **Brand Personality** - Core attributes and personality statement
3. **Voice Attributes** - Detailed attribute definitions with examples
4. **Tone Matrix** - Context-based tone variation guide
5. **Visual Identity** - Visual language guidelines
6. **Channel Guidelines** - Channel-specific adaptation rules
7. **Quick Reference** - One-page cheat sheet for daily use
8. **Glossary** - Approved terminology and usage standards

### 2. Compile Executive Summary
Draft a concise executive summary that covers:
- Why this brand voice framework exists
- Who should use this document
- How to use this document
- Key principles at a glance
- Review the summary with the user

### 3. Assemble Full Document
Compile all sections from previous steps into the structured document:
- Ensure consistent formatting throughout
- Add cross-references between sections where relevant
- Include the examples and do's/don'ts from each step
- Verify no sections have gaps or placeholder content

### 4. Create Quick Reference Card
Build a one-page quick reference that includes:
- Voice attributes at a glance
- Top 5 do's and don'ts
- Tone matrix summary (simplified)
- Key phrases and terminology
- Where to find the full guidelines

### 5. User Review
Present the complete document to the user for review:
- Walk through each section highlighting key decisions
- Ask for any final adjustments or additions
- Confirm the document meets their expectations
- Note any items that need future refinement

### Save Progress
Save the compiled guidelines document to the working output file.
