# Step 5: Complete

## STEP GOAL:
Finalize the social content package, save the deliverable, and provide guidance on scheduling, monitoring, and iteration.

## MANDATORY EXECUTION RULES:
- NEVER generate content without user input
- YOU ARE A FACILITATOR, not a content generator
- FORBIDDEN to look ahead to future steps
- WAIT for user confirmation before proceeding

## Sequence of Instructions

### 1. Final Review Summary
Present a summary of everything created:
- Number of platforms strategized
- Content pillars defined
- Calendar duration and post count
- Total posts drafted per platform
- Visual assets needed
- Compliance elements integrated

### 2. Save Deliverable
Save the final document:
- **Filename:** `social-content-{project_name}-{date}.md`
- **Location:** `{project-root}/{output_folder}/campaign-artifacts/`
- Confirm the file has been saved successfully

### 3. Implementation Checklist
Provide a launch checklist:
- [ ] Submit all posts for compliance review
- [ ] Create or source all visual assets
- [ ] Optimize all profile pages and bios
- [ ] Schedule posts using social management tool
- [ ] Set up UTM tracking for all links
- [ ] Configure social listening for brand mentions
- [ ] Prepare engagement response guidelines for team
- [ ] Set up analytics dashboards per platform
- [ ] Establish weekly reporting cadence
- [ ] Plan content calendar refresh cycle

### 4. Performance Metrics to Track
Define key metrics:
- Reach and impressions per post
- Engagement rate (likes, comments, shares)
- Click-through rate to offering page
- Follower growth rate
- Conversion from social to investor sign-up
- Best performing content types and themes
- Optimal posting times based on data

### 5. Next Steps Recommendations
Advise the user:
- Submit all posts for compliance review before scheduling
- **Create Email Campaign**: Coordinate email and social messaging
- **Create Landing Pages**: Ensure social traffic has optimized destinations
- Schedule weekly content reviews and calendar updates
- Plan for real-time engagement and community management
