# Step 8: Complete

## STEP GOAL:
Finalize the financial content package, save the deliverable, and provide guidance on review, distribution, and ongoing updates.

## MANDATORY EXECUTION RULES:
- NEVER generate content without user input
- YOU ARE A FACILITATOR, not a content generator
- FORBIDDEN to look ahead to future steps
- WAIT for user confirmation before proceeding

## Sequence of Instructions

### 1. Final Review Summary
Present a summary of everything created:
- Financial data collected and verified
- Narrative framework established
- Key metrics dashboard defined (count of metrics)
- Investor FAQ completed (count of Q&A pairs)
- Performance summary drafted
- Forward guidance developed
- Compliance elements integrated throughout

### 2. Save Deliverable
Save the final document:
- **Filename:** `financial-content-{project_name}-{date}.md`
- **Location:** `{project-root}/{output_folder}/campaign-artifacts/`
- Confirm the file has been saved successfully

### 3. Critical Reminders
Remind the user:
- All financial content must be reviewed by qualified financial professionals
- Financial projections are particularly sensitive and should be reviewed by counsel
- Ensure consistency between financial content package and offering document
- Any material changes in financial performance require content updates
- Non-GAAP metrics require proper reconciliation and disclosure

### 4. Distribution Guide
Advise on how to use the financial content package:
- **Offering Circular**: Financial narrative and data for the offering document
- **Investor Deck**: Financial slides and key metrics
- **Email Campaigns**: Financial highlights and milestone updates
- **Landing Pages**: Key metrics and FAQ for web content
- **Investor Updates**: Performance summary template for periodic updates
- **Sales Team**: FAQ and talking points for investor conversations

### 5. Update Cadence
Recommend an update schedule:
- Monthly: Key metrics dashboard refresh
- Quarterly: Performance summary and guidance update
- As-needed: FAQ updates based on investor questions
- Material events: Immediate financial content refresh

### 6. Next Steps Recommendations
Advise the user:
- Submit to financial advisor and legal counsel for review
- **Create Investor Deck**: Integrate financial narrative and metrics
- **Create Email Campaign**: Use financial highlights in investor outreach
- **Create Offering Circular**: Financial content feeds directly into offering document
- Schedule the first periodic update

### 7. Workflow Complete
- Mark all steps as completed
- Display completion confirmation
- Return to agent menu
