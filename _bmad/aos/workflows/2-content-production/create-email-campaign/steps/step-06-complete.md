# Step 6: Complete

## STEP GOAL:
Finalize the email campaign package, save the deliverable, and provide guidance on implementation and testing.

## MANDATORY EXECUTION RULES:
- NEVER generate content without user input
- YOU ARE A FACILITATOR, not a content generator
- FORBIDDEN to look ahead to future steps
- WAIT for user confirmation before proceeding

## Sequence of Instructions

### 1. Final Review Summary
Present a summary of everything created:
- Number of audience segments mapped
- Number of email sequences designed
- Total emails drafted
- Compliance check results
- Personalization elements defined
- Behavioral triggers configured

### 2. Save Deliverable
Save the final document:
- **Filename:** `email-campaign-{project_name}-{date}.md`
- **Location:** `{project-root}/{output_folder}/campaign-artifacts/`
- Confirm the file has been saved successfully

### 3. Implementation Checklist
Provide a deployment checklist:
- [ ] Upload email content to ESP platform
- [ ] Configure automation sequences and triggers
- [ ] Set up merge fields and personalization
- [ ] Configure segmentation and targeting
- [ ] Set up A/B tests for subject lines
- [ ] Send test emails to internal team
- [ ] Review test emails on desktop and mobile
- [ ] Verify all links and CTAs work correctly
- [ ] Confirm unsubscribe mechanism works
- [ ] Schedule sends or activate automation
- [ ] Set up tracking and analytics dashboards

### 4. Testing Recommendations
Advise the user on pre-launch testing:
- Send to seed list across email clients (Gmail, Outlook, Apple Mail)
- Check mobile rendering
- Verify personalization fields populate correctly
- Test all links and CTA buttons
- Verify compliance elements display properly
- Spam filter testing

### 5. Next Steps Recommendations
Advise the user:
- **Create Landing Pages**: Build destination pages for email CTAs
- **Create Social Content**: Coordinate social with email messaging
- Monitor open rates, click rates, and conversion after launch
- Schedule performance review after first sequence completes

### 6. Workflow Complete
- Mark all steps as completed
- Display completion confirmation
- Return to agent menu
